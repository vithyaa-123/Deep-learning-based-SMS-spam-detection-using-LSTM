# Deep-learning-based-SMS-spam-detection-using-LSTM
This project focuses on detecting spam messages using a Deep Learning model (LSTM). It classifies SMS messages into Spam or Ham (Not Spam) using Natural Language Processing (NLP) techniques

**Technologies Used**
Python
Pandas, NumPy
Scikit-learn
TensorFlow / Keras
KaggleHub

**Dataset**
Dataset used: SMS Spam Collection Dataset
Source: Kaggle
Automatically downloaded using kagglehub
